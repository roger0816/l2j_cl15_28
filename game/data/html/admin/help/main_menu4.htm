1-主面板按鈕
1-3種改變目標命運的方法。
Kill: 殺死目標，可選擇以QB值作為半徑或玩家名稱。
Res: 復活目標，可選擇以玩家名稱或指定半徑內的玩家為對象。
Open: 假設目標是一扇門，它會在您的命令下打開。
Close: 假設目標是一扇門，它會在您的命令下關閉。
Recall NPC: 假設目標是一個NPC/Mob，它將從現在開始在您當前的座標和面向上生成。
Delete: 假設目標是一個NPC/Mob，它將不會再次生成。
Heal: 像//kill或//res一樣，選擇目標、名稱或半徑，並將其HP/MP/CP一次性恢復到最大值。
Full Food: 對於有問題的寵物很有用，允許寵物恢復其食物水平並正確地取消召喚。
