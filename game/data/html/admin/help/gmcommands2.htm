//changelvl - 更改目標玩家的訪問級別。

//ban_acc - 用於踢出和封禁指定玩家的帳戶。
用法：//ban_acc kadar
//ban_char - 用於踢出和封禁指定角色。
用法：//ban_char kadar
//ban_chat - 禁言一個玩家。
用法：//ban_chat kadar [持續時間]
//unban_acc - 用於解封指定玩家的帳戶。
用法：//unban_acc kadar
//unban_char - 用於解封指定角色。

//unban_chat - 解除一個玩家的禁言狀態。
