GM指令頁面：123

這些是遊戲中可用的GM指令。

//admin - 打開管理員菜單。
//gmchat - 發送只有GM可以看到的全局消息。
用法 - //gmchat 在此輸入消息
//gm - 關閉你的GM狀態並將訪問級別設置為0。
//invul - 打開或關閉你的無敵狀態。
//delete - 用於刪除目標Mob。Mob將不會重生。
//kill - 用於殺死目標Mob。Mob將重生。
//target - 用於定位Mob或玩家。
用法：//target kadar
//buy - 打開GM商店。
//gmshop - 打開GM商店。
//announce_menu - 打開公告菜單。
//list_announcements - 列出當前公告。
//reload_announcements - 重新加載公告列表。
//announce_announcements - 張貼列表中的所有公告。
//add_announcement - 添加公告。
用法：//add_announcement 在此輸入您的公告
//del_announcement - 刪除公告。
用法：//del_announcement 在此輸入您的公告
//announce - 張貼公告。
用法：//announce 在此輸入您的公告
//itemcreate - 打開物品創建菜單。
//create_item - 創建物品。
用法：//create_item 物品ID [數量]
//server_shutdown - 開始關閉服務器的過程。
用法：//server_shutdown 秒數
//server_restart - 開始重啟服務器的過程。
用法：//server_restart 秒數
//server_abort - 停止服務器關閉。
//show_spawns - 打開生成菜單。
//spawn - 在當前目標上生成Mob。將在服務器重啟後重生
用法：//spawn npc_id
//spawn_once - 在當前目標上生成Mob。不會再次重生。
用法：//spawn_monster npc_id
//spawn_index - 顯示具有相應級別的怪物菜單。
用法：//spawn_index 級別
//show_skills - 列出目標玩家的技能。
//remove_skills - 刪除目標玩家的技能。
//skill_list - 打開技能菜單。
//skill_index - 根據玩家的統計數據打開技能菜單。
//add_skill - 將技能添加到目標玩家。
用法：//add_skill 技能ID 等級
//remove_skill - 從目標玩家中刪除技能。
用法：//remove_skill 技能ID
//get_skills - 暫時給予您目標玩家的技能。
//reset_skills - 將您的技能恢復到使用get_skills命令之前的狀態。
//add_exp_sp_to_character - 打開目標玩家的經驗值和SP菜單。
//add_exp_sp - 將經驗值和SP添加到目標玩家。
用法：//add_exp_sp 經驗值數量 SP數量
//remove_exp_sp - 從目標玩家中刪除經驗值和SP。
用法：//remove_exp_sp 經驗值數量 SP數量
//edit_character - 打開編輯目標玩家的菜單。
//current_player - 打開角色列表。
//character_list - 打開在線玩家列表。
//show_characters - 打開在線玩家列表。
//find_character - 打開查找玩家菜單。
//save_modifications - 保存對目標玩家的編輯。
//show_moves - 打開傳送菜單。
//show_moves_other - 打開名為“其他”的傳送菜單。
//show_teleport - 打開傳送菜單。
//teleport_to_character - 將您傳送到指定的玩家。
用法：//teleport_to_character kadar
//teleportto - 將您傳送到指定的玩家。
用法：//teleportto kadar
//move_to - 將您傳送到指定的坐標。
用法：//move_to 1111 2222 3333
//teleport_character - 將目標玩家傳送到指定的坐標。
用法：//move_to 1111 2222 3333
//recall - 召喚指定的玩家到您的位置。
用法：//recall kadar
//restore - 恢復損壞的玩家信息。
//repair - 修復損壞的玩家信息。
