怪物群組控制指令

//mobgroup_create [groupId] [npcId] [number]
創建一個包含[number]個[npcId]類型怪物的群組，群組ID為[groupId]。
使用此命令沒有視覺效果，群組成員需要生成才能變為活動狀態。

//mobgroup_remove [groupId]
刪除群組[groupId]，同時取消生成任何仍然存活的群組成員。

//mobgroup_spawn [groupId] [locx] [locy] [locz]
生成群組[groupId]的成員。如果未指定位置，則群組將在GM周圍生成。
默認情況下，怪物處於閒置狀態，等待進一步命令。
//mobgroup_unspawn [groupId]
取消生成群組[groupId]的成員。群組不會被刪除，可以再次使用//mobgroup_spawn命令生成。
//mobgroup_list ([groupId])
列出所有已創建的怪物群組。如果指定了[groupId]，則顯示該群組成員的列表。

其他指令
//mobgroup_follow [groupId] - 群組[groupId]將跟隨GM。
//mobgroup_return [groupId] - 群組[groupId]將返回GM。
//mobgroup_teleport [groupId] ([playerName]) - 群組[groupId]將被傳送到並跟隨指定的玩家（如果未指定，則跟隨GM）。
//mobgroup_attack [groupId] - 群組[groupId]將攻擊GM當前的目標（如果有）。
//mobgroup_rnd [groupId] - 群組[groupId]將攻擊隨機選擇的目標。
//mobgroup_attackgrp [groupId1] [groupId2] - 群組[groupId1]將攻擊[groupId2]中的成員。
//mobgroup_casting [groupId] - 群組[groupId]的怪物將只使用技能進行攻擊。
//mobgroup_nomove [groupId] [on/off] - 切換群組[groupId]的怪物不再移動。與僅使用技能模式一起使用時很有趣。
//mobgroup_invul [groupId] [on/off] - 切換群組[groupId]的怪物為無敵狀態。
//mobgroup_idle [groupId] - 將群組[groupId]中的怪物返回到閒置（等待）狀態。
//mobgroup_kill [groupId] - 消滅群組[groupId]中所有存活的怪物。
