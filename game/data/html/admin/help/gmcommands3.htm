//para_all - 使所有玩家麻痺
//unpara_all - 解除所有玩家的麻痺
//para - 麻痺目標
//unpara - 解除目標的麻痺
//bighead - 讓目標的頭變大
//shrinkhead - 讓目標的頭恢復正常大小
//res - 復活目標、玩家名稱或半徑內的角色
//mammon_find - 列出Mammon NPC的當前位置（如果有）
//mammon_respawn - 列出Mammon NPC的當前重生時間（如果有）

//list_spawns <npcId> <teleIndex> - 根據給定的生成點索引（使用//list_spawns <npcId>Wink）將玩家傳送到玩家。
//gmliston - （將GM添加到gmlist）
//gmlistoff - （從gmlist中刪除GM）
//silence - （啟用/禁用私人消息模式）
//diet - 玩家的飲食模式當前狀態。
//diet on - 啟用飲食模式。
//diet off - 禁用飲食模式。
//tradeoff - 玩家的交易當前狀態。
//tradeoff on - 啟用與其他玩家的交易。
//tradeoff off - 禁用與其他玩家的交易。

//heal <name> - 治療名為<name>的角色。不需要選擇目標。
//give_all_skills - 將所有可在其等級和職業中學習的技能授予目標
//openall - 打開所有門
//closeall - 關閉所有門
//unspawnall - 從世界中刪除所有NPC
//respawnall - 重新加載NPC數據
//reload skill - 重新加載技能數據。
//reload multisell - 重新加載多重出售數據。

//atmosphere <sky> <day|night> - 將當前天空設置為白天或黑夜。
//quest_reload <questFolder> or <id> - 重新加載所選任務。
//cache_htm_rebuild - 重建並重新加載HTML緩存。
