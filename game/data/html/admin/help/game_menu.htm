2-遊戲面板按鈕
 L2的特定內容包括：角色、誓言、詛咒武器和靜態事件。
2-1 手中的玩家。
管理：常見的與玩家相關的操作遺產面板。
列表：顯示在線玩家列表的對話框，顯示等級、職業和搜索框。點擊名字查看詳細信息。
設置名稱：快速永久更改玩家的名稱。旨在執行規則，而不是用於事件。
設置稱號：快速永久更改稱號。如果玩家是貴族或屬於一個氏族，他們可以稍後更改它。
2-2 氏族：合成社會的中心。
創建：目標角色將成為領袖。必須在快速框中提供氏族名稱。
刪除：分而治之。通過瞄準其領袖，立即解散一個氏族。
設置等級：指定從0到8的等級。目標必須是氏族領袖。
聲望：指定正值或負值以改變當前聲望分數。目標必須是氏族領袖。
城堡/城堡大廳：城堡、攻城戰和氏族大廳管理對話框。
信息：顯示誓言信息窗口。目標必須是氏族領袖。
2-3 詛咒武器。混沌自選。
信息：顯示所有支持的CW的狀態信息。
添加：如果目標是角色，將收到指定的武器。項目ID、項目名稱或其一部分必須在QB中寫入（例如：zariche）
刪除：從世界上取走詛咒武器。項目ID、項目名稱或其一部分必須在快速框中指定（例如：akamanah）
2-4 七個徵兆和移動的矮人。
鐵匠：打印坐標並將您帶到它（如果它正在生成）。
商人：打印坐標並將您帶到它（如果它正在生成）。
重生：顯示重生ETA作為系統消息。
2-5 奧林匹克運動會：YAWIP。
保存奧林匹克：手動保存奧林匹克狀態。
手動英雄：計算贏家，獎勵新英雄。
