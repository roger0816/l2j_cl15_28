1-主面板按鈕
1-2 快速處理玩家的愚蠢行為。
ChatBan: 可以從QB中取得玩家名稱和持續時間，或只是目標的聊天禁止時間。
ChatUnBan: 目標或玩家名稱被解除了chan禁止。
Jail: 像聊天禁止一樣工作，但是將角色關進監獄。如果沒有指定持續時間，懲罰將永遠持續。
UnJail: 讓玩家離開監獄，他們會出現在Floran村莊。離線玩家也會受到影響。
Kick: 斷開玩家與伺服器的連接。
Ban: 在QB中給出角色名稱，其整個帳戶都會被禁止。
UnBan: 給出玩家名稱，其帳戶將被解禁。
Find: 顯示與您指定的任何內容匹配的角色名稱列表，或者如果沒有，則顯示列表/查找對話框。
Repair: 為您提供的角色名稱切換回安全默認值的某些值。明智使用！
