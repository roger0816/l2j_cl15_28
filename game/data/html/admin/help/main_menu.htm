面板幫助：
0-介紹
上方側邊欄包含主要管理區域。此欄跨越指向五個主要頁面。
從//admin1到//admin5的命令將直接帶您到相關的管理區域。
一般來說，您可能更常使用的命令按鈕應該首先出現。
每當命令按鈕需要一個或多個參數時，您應該在快速框中用空格分隔它們。
一些按鈕將在您的目標上工作，其他按鈕將僅考慮參數，而其他一些按鈕將需要兩者。
通常，依賴快速框的命令應該更靠近它。
每個重要頁面都有一個快速框。幫助按鈕包含上下文相關的提示，請嘗試它們全部；）
當使用不正確的目標和/或不正確/缺少的參數調用某些命令時，一些與操作相關的對話框將彈出。
其他命令則通過發送系統消息顯示使用指南，請確保您的客戶端已正確設置以查看這些消息！
