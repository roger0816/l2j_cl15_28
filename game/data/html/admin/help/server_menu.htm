4-伺服器面板按鈕
 這裡是與伺服器相關的操作：設置、倍率、快取。
4-1重新載入和其他。
重新載入：嘗試重新載入指定的實例管理器。可能的值為：multisell、skill、npc、htm、item、instancemanager、npcwalkers。
重新載入 HTM：清除 HTM 快取。
重新載入檔案：清除指定 HTM 檔案的快取。
技能測試：類似於 //effect，接收技能 ID 作為參數，影響目標或施法者。
重新載入 CW：重新載入詛咒武器管理器。
重新載入區域：重新載入區域管理器。
重新載入紋章：重建紋章快取。
紋章修復：如果發現舊的紋章檔案，則轉換它們。
重新載入傳送點：重新載入傳送目的地。
社交：讓角色播放社交動畫。快速框可以包含社交 ID、（可選）玩家名稱和（可選）半徑。如果只提供一個參數，則影響目標。
全部開啟/關閉：開啟或關閉伺服器中的所有門。
區域檢查：按需執行區域檢查。

（知道自己在做什麼的人，也知道哪個是有效的）
4-3開發。
戰鬥計算：啟動兩個怪物之間的戰鬥計算對話框。 封包製作：啟動封包製作設施。
系統訊息：將指定的系統訊息發送給您。
4-4維護。
統計：顯示有關記憶體使用、任務等方面的伺服器統計資訊。
登錄：顯示與登錄伺服器相關的信息和選項。
關閉伺服器：顯示重新啟動/關閉伺服器相關的操作。
