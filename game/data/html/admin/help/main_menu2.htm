1-主要面板按鈕
這裡整理了最常用的指令。
1-1 你最常使用的指令
物品: 從快速框中輸入物品ID和數量，可以在你的背包中創建該物品。如果未指定，則顯示特定對話框。
傳送: 從快速框中讀取座標。如果未指定，則顯示傳送和移動菜單。
生成: 輸入NPC ID或名稱（空格必須改為下劃線），可選生成數量和重生時間。如果未指定，則顯示特定對話框。
列出生成: 輸入NPC ID或名稱（空格需要替換為下劃線）。將結果顯示為系統消息。
公告: 從快速框中輸入要公告的文本；如果未指定文本，則顯示公告菜單。
設置: 快速框中需要“參數=值”。如果未指定有效內容，則顯示設置對話框。
GM聊天: 僅向GM廣播快速框內容。廣播後會彈出管理菜單。
列出+前往: 像Google的Lucky按鈕一樣，將傳送您到List Spawn命令的第一個結果。
前往: 從快速框中讀取玩家名稱。如果未指定，則顯示角色管理屏幕。
召回: 如果在快速框中指定了玩家名稱，則將該玩家帶到您的位置。
業障: 將目標的業障點數設置為快速框中指定的值（如果目標是玩家，當然）。
無業障: 就像業障按鈕一樣作用於您的目標，但忽略QB值，將業障設置為零。
