3-特效面板按鈕
 L2特效在這裡：地震、天空變化、奇怪的角色和怪物動畫。
3-1 第一批特效。
PolyMorph：讓目標（玩家）看起來像指定的怪物。
Unpoly：取消指定目標可能有的變身效果。
PolySelf：將自己變身為指定的npcId，只會看到標題更改。
UnPolySelf：取消任何可能有的變身效果。
Para：使角色（目標/名稱）癱瘓。
UnPara：取消癱瘓效果（目標/名稱）。
ParaAll：使世界上除了你以外的任何角色癱瘓。請明智使用，會惹惱人並導致延遲。
UnParaAll：取消每個在線角色的癱瘓效果。
Effect：讓角色播放施法動畫。快速框應包含技能ID，（可選）技能等級和（可選）持續時間。
Social：讓角色播放社交動畫。快速框可以包含社交ID，（可選）玩家名稱和（可選）半徑。如果只提供一個參數，則影響目標。
Abnormal：改變角色的異常狀態。快速框可以包含異常位掩碼，（可選）玩家名稱和（可選）半徑。如果只提供一個參數，則影響目標。
Transform：進行卡瑪埃爾變身。快速框必須包含transformId，並影響目標。
3-2 第二批特效（較小的目標相關）。
Day/Night Sky：天空變化。影響整個世界。
Dawn/Dusk Sky：強制執行七個徵兆的天空。
Red Sky：強制執行詛咒武器天空效果。
Big/Shrink Head：0x2000異常的快捷方式。僅影響目標。如需進一步控制，請參見異常。
Quake：引起地震。快速框應包含強度和持續時間。
SetName：暫時更改角色名稱。
Sounds：打開一個帶有環境聲音的菜單供您播放。
Mob Groups：打開一個與MobGroup相關的選項菜單。
