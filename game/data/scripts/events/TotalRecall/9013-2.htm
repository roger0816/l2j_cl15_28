青蛙：
一個怪物偷走了我製作的愛情藥水！那是我的心血和汗水！
喝一口它就可以讓你和你的隊友的物理攻擊/魔法攻擊/速度/施法速度翻倍2分鐘！很棒，對吧？
我不會稱這是副作用，但這個藥水確實會稍微改變你的武器外觀。但不用擔心，因為它不會變得醜陋！記得我是愛情的使者嗎？
（愛情藥水的效果在重新啟動遊戲時會消失。）
此外，你可以選擇巧克力松露、糖果心臟或安慰曲奇。每種藥水在和平區域可以恢復你的活力，並且也有自己的特殊能力。
巧克力松露讓你使用放縱技能，增加STR + 2、INT + 2和屬性抗性 + 20。
糖果心臟讓你使用糖果衝刺技能，增加STR + 2、INT + 2、DEX + 2和WIT + 2。
安慰曲奇讓你使用永遠孤獨技能，增加STR + 2、INT + 2、CON + 2和MEN + 2。如果有3個人每個人都使用這些物品聚集在一起，他們可以獲得所有效果，所以你可能想去尋找新朋友！這些技能是從愛情藥水的剩餘物製作的，所以它們可能看起來有點，呃，過度了...但是，它們很可愛，我保證！
哦，你知道變身和召喚物品嗎？
變身物品讓你變成雪人、稻草人傑克、錫魔像、臨時蝙蝠、生氣的毛巾、淑女毛巾或昏昏欲睡的毛巾。它們有什麼好處，你問？嗯，它們很可愛！超級可愛！
